# Task Manager Pro

Projeto Flutter com Material 3, CRUD SQLite e navegação push/pop.

## Como rodar
1. No diretório do projeto:
   ```bash
   flutter create .
   flutter pub get
   flutter run -d emulator-XXXX
   ```
2. O banco SQLite é criado no primeiro uso.
